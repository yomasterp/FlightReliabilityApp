Flight Reliability & Availability Tracker   
Uses the free Aviationstack API to collect any available flight status data, then analyze airline reliability, and predict other valuable insights.

---

Project Overview
This project uses real-time and historical flight data from Aviationstack, stores raw JSON responses, and processes them to analyze:

- Airline on-time performance  
- Delay trends  
- Availability patterns  
- Potential predictive indicators for reliability  

The long-term goal is to predict future reliability and availability for selected airlines.
